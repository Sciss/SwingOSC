This class allows you to add any JComponent or JPanel subclass to an JSCWindow or one of its

containers. last mod: 03-oct-06 sciss.

superclass: [JSCView], [JSCContainerView]

see also: [JavaObject]


JSCPlugView, JSCPlugContainerView


Warning: as of v0.42, the constructor signature changed. old:


*new( (JSCView) parent, (Rect) bounds, (SwingOSC) argServer, (Integer) id, (Symbol) javaClass );


new:


*new( (JSCView) parent, (Rect) bounds, (JavaObject) javaObject );


the server is taken from javaObject.



Note: There is no getter methods implementation right now
