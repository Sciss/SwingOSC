JSCScrollPane

Warning: still experimental! Note: please use the abstraction layer GUI.scrollPane if possible! (see GUI).

A container wrapper that adds horizontal and/or vertical scrolling bars if the bounds of its child extend the scrollpane's own bounds. There can be only one child at the moment, this is usually a JSCCompositeView or similar. 

(
    w = JSCWindow.new;
    t = JSCScrollPane( w, w.view.bounds );
    t.resize = 5;
    c = 0.5.coin.if( JSCCompositeView, JSCVLayoutView ).new( t, Rect( 0, 0, 380, 900 ));
    60.do({ arg i;
        JSCButton( c, Rect( rrand( 20, 300 ), i * 30 + 10, 75, 24 ))
            .states_([[ "Start "++i, Color.black, Color.rand ],
                     [ "Stop "++i, Color.white, Color.red ]]);
    });
    w.front;
)

// how the scrollbars are shown
t.verticalScrollBarShown   = \never;
t.verticalScrollBarShown   = \always;
t.verticalScrollBarShown   = \auto; // shown when child exceeds viewport size, otherwise hidden
t.horizontalScrollBarShown = \never;
t.horizontalScrollBarShown = \always;
t.horizontalScrollBarShown = \auto;
